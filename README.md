# DSTP 3.0 Assignment 1

A Python assignment demonstrating fundamental programming concepts, including conditional statements, data structures, and Object-Oriented Programming (OOP).

## Project Overview

This assignment contains three programming tasks implemented in a Jupyter Notebook:

1. **Tax Calculation System** - Calculate income tax based on salary ranges
2. **Word Length Dictionary** - Map words to their corresponding lengths
3. **Chat System using OOP** - A complete chat application with User, Message, and ChatRoom classes

## Contents

### Assignment1.ipynb

The notebook contains all solutions with explanations and demonstrations:

#### Task 1: Tax Calculation
- Calculates progressive tax based on salary brackets
- Salary < 30,000: 5% tax
- Salary 30,000-70,000: 15% tax
- Salary > 70,000: 25% tax

#### Task 2: Word Length Dictionary 
- Creates a dictionary mapping words to their lengths
- Uses Python dictionary comprehension
- Example: `{"apple": 5, "banana": 6, "kiwi": 4, "cherry": 6, "mango": 5}`

#### Task 3: Chat System using OOP
- **User Class**: Represents a user with ID and username
- **Message Class**: Stores message content, sender, and timestamp
- **ChatRoom Class**: Manages users, messages, and chat history
- **Features**:
  - Users can join and leave chat rooms
  - Send and receive messages
  - View complete chat history
  - Track active users in real-time
